# Simpson Invasion

Classic alien invasion game with the Simpson icons...

## Description
![](images/game_img.png)
Move left and right, Use spacebar to shoot.
You got 3 lives shows on top left.
The Score and level shows on the top right.

## Getting Started

### Dependencies

Windows10

### Installing

Download and unzip the file

### Executing program

Run main.py
